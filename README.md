ffmpeg-tsmuxer
============

Mux h.264 raw stream to TS based on FFmpeg.

## Features
* Mux h.264 raw stream to TS based on FFmpeg.
* Use cmake for cross-platform build.

## Dependencies
* libavformat
* libavcodec
* libavutil

## Build
* `./start_build.sh`
* build target is located in build/ffmpeg_tsmuxer

## Usage
* `ffmpeg_tsmuxer`
